# RAG Concepts - Hands-On Learning Labs

A comprehensive collection of Jupyter notebook labs for learning **Retrieval-Augmented Generation (RAG)** systems from scratch. These practical, hands-on exercises guide you through building mini RAG systems using OpenAI API, OpenRouter, and FAISS vector database.
